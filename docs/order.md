## 1. 초기세팅

현재 프론트 개발 기술 스텍을 먼저 확정을한다 되도록이면 vercel에서 쉽게 배포가능한걸로

그리고 서 확정된 기술스택으로 초기 세팅을 한다 

git remote add origin https://github.com/hefxpzwk/wwt-client.git
git branch -M main
git push -u origin main

여기에 연결을 한다 커밋 푸시를 한다.
